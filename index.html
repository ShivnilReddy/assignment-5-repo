How to Apply CSS to HTML

To include an external CSS file, use the <link rel="stylesheet" href="path/to/style.css"> tag in the <head> of your HTML document.

How to Use JavaScript in a Webpage

Link to an external JavaScript file using <script src="path/to/script.js"></script> at the end of your HTML document, before the closing </body> tag. Alternatively, you can include JavaScript directly within HTML using <script> tags.
